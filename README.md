# Spring Introduction Study（キム・ヨンハン／Inflearn ベース）

Spring の概念を整理し、実習コードを記録するリポジトリです。  
**個人学習用（非公式）**であり、講座／プラットフォームとは直接の関係はありません。

---

## 目的（Goals）
- Spring Web 開発の基礎
- 会員管理 例（**バックエンド**）
- **Spring Bean** と依存関係（DI/コンテナ）
- 会員管理 例 — **Web MVC** 開発
- Spring **DB アクセス技術**
- **AOP**
- 実習サンプルと整理ノートの併用管理

## 環境（Environment）
- **Build**: Gradle（**Groovy DSL**）
- **Spring Boot**: 3.4.4
- **Java**: 17
- **IDE**: IntelliJ IDEA
- **Dependencies**: Spring Web, Thymeleaf
